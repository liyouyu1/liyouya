# 猫猫糕桌宠素材

这个目录保存桌宠使用的素材与后续扩展位。

## 当前结构

- `pets/`: 首版可切换的猫猫糕基础贴纸，使用 WebP 格式。
- `pets/manifest.json`: 桌宠加载清单，包含默认猫猫糕、显示名、文件名和标签。
- `expressions/`: 预留后续表情素材。
- `actions/`: 预留后续动作素材。
- `voice/`: 预留后续语音素材。
- `icons/`: 预留应用图标与托盘图标。

## 首版素材

当前贴纸原始文件是带透明通道的 WebP。早期文件扩展名写作 `.png`，实现时已规范化为 `.webp`，避免运行时图像库误判格式。

默认启动角色是 `冰糕`，对应 manifest 中的 `bing_gao`。

## 后续素材规范

- 贴纸建议保持透明背景。
- 基础贴纸放入 `pets/`，并同步更新 `pets/manifest.json`。
- 表情差分放入 `expressions/`。
- 动作序列或短动画帧放入 `actions/`。
- 语音文件放入 `voice/`，后续由 agent 层按动作或文本触发。
